= Meetup (tentative name)

Meetup is for collecting pictures of attendees of parties.

== Model
* User
  * [has_many] albums
* Album
  * name
  * [has_many] pictures
  * control
* Picture
  * paperclip
  * attendee
  * facebook link
  * twitter link
  * instagram link
